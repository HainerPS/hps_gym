# 🏋️ HPS Gym Website

This project is a fitness/gym website developed to practice front-end development concepts, focusing on layout structure, user experience, and visual presentation.

## 📌 About the project

HPS Gym is a multi-section website designed to present a gym's services, environment, and contact information in a clear and organized way.

The goal of this project is to simulate a real-world business website, applying HTML and CSS concepts to create a structured and visually consistent interface.

## 🚀 Live Preview

Access the project online:  
👉 https://hps-gym.vercel.app/

## 📄 Features

- Responsive layout structure
- Navigation menu between sections
- Presentation of services and training programs
- Contact section
- Visual hierarchy using HTML and CSS
